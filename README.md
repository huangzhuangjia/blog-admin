# iview-admin-template

> 一套后台管理系统模板，集成了Vue2.5全家桶 + TypeScript + UI框架 iview + json-server

## Usage

### Installation

> $ npm install

### serve with hot reload

> $ npm run dev

默认开启8081端口

### serve with mock api

> $ npm run devMock

项目开发阶段用json-server模拟后台数据，默认开启端口3000

> 注： 运行dev命令使用后台代理接口，运行devMock命令使用本地json-server进行模拟接口
